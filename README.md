# Dashboard – Aversión al Riesgo en Colombia

Este tablero fue construido con Streamlit y replica el modelo de aversión al riesgo 
de Chávez, Milanesi & Pesce (2021) aplicado al mercado colombiano.

Incluye estimaciones por:
- CRRA
- FTP
- GARCH

Para ejecutarlo en Streamlit Cloud:
1. Subir este repositorio a GitHub
2. Crear app en https://streamlit.io/cloud
3. Seleccionar app.py como archivo principal
